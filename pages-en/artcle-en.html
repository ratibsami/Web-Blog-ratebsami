Adobe XD Comprehensive Learning Course

Wednesday, March 13, 2024

The powerful Adobe XD software was released for free as an initial version on April 10th, 2016. This software was created for designing flexible user interface (UI) and user experience (UX). I, Ratib Sami, a graphic designer, will teach you Adobe XD software from scratch in this course. Throughout this course, you will become familiar with the following concepts:

  • Introduction to basic principles of UI design
  • Introduction to fundamental concepts of UX
  • Creating initial prototypes
  • Comprehensive review of the best designs worldwide
  • Creating a mini project for a clothing store

By the end of this course, you will be fully acquainted with UI and UX design and will be able to create large projects. To access all the videos, click here.
